Basic Commands:
``!song``, ``!album`` (/) : Picks a random Taylor Swift song or album
``!8ball`` (/) : Gives a random answer to a yes or no question
``!yesno`` : Adds yes, no, maybe react for a simple yes/no poll
``!choose [one, two, three]`` (/) : Chooses one of several options randomly
``!namemix`` (/) : Mixes two parts of server names
``!randmember`` : Selects a server member at random who has spoken in the last day.
``!shitpost [optional number]`` (/) : Posts a shitpost
``!acronym`` (/): Looks up a Taylor Swift song acronym

Saves:
``!rg`` (/save random): Gets a saved item at random
``!g [keyword]``, ``!get [keyword]`` (/) : Retrieves a saved item
``!gmeta [keyword]`` : View info about a saved item
``!save [keyword] [link/text]`` : Creates a new saved item
``!delete [keyword]`` (/save delete) : Deletes an item you saved

Stats:
``!seen [user]`` : Shows when a member was last active
``!activity`` (/) : Gives a link to charts of your server activity stats
``!totals`` (/) : Shows message stats by channel
``!lorpoints [user]``, ``!words [user]`` (/): View word or lorpoint stats
``!awards [user]`` (/): View awards for user
``!rankwords``, ``!ranklorpoints`` : Ranks members by words or lorpoints

Questions (DMs only):
``!question [question]`` (/question create) : Creates a new question
``!answer [id] [answer]`` (/) : Submits an answer to a question
``!getanswers [id]``, ``!getnewanswers [id]`` (/question get-answers): View answers to a question you created
``!watch [id]``, ``!unwatch [id]`` : Turns on/off notifications for new answers to your question
``!closequestion [id]``, ``!openquestion [id]`` (/question close, /question open) : Closes/opens responses to your question


Bot Info:
``!fversion`` (/) : Displays the bot version
``!fhelp`` (/) : Displays this message.
